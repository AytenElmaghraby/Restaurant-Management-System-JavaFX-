# 🍽 Restaurant Management System

A JavaFX desktop application designed to simplify restaurant reservation and food ordering.

---

## 📖 Overview

This project provides a user-friendly graphical interface for managing restaurant operations including reservations, customer information, and orders.

---

## ✨ Features

- Customer Management
- Table Reservation
- Food Ordering
- Order Management
- Database Integration
- Exception Handling
- Interactive JavaFX Interface

---

## 💡 OOP Concepts

- Inheritance
- Polymorphism
- Abstraction
- Interface
- Composition
- Aggregation
- Association
- ArrayList
- Exception Handling

---

## 🛠 Technologies

- Java
- JavaFX
- MySQL
- JDBC

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AytenElmaghraby/Restaurant-Management-System.git
```

Import the project into NetBeans.

Import the SQL database.

Run the application.
